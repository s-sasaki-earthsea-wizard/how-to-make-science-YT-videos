# 3-1: アウトプットがインプットを変える

> 動画コンセプトは [instructions/video_concept.md](../../../instructions/video_concept.md) を参照

**この動画のテーマ**: アウトプット前提のリサーチマインドセット

## アウトライン

### 1. Hook（0:00-0:20）

- 結論を先出し：**アウトプットを前提にすると、インプットが変わる**
- 「教科書や論文を読むのが苦手」「何を調べればいいかわからない」と悩む必要もありません

### 2. Before/After（0:20-1:00）

**YouTube始める前**

- 情報を「消費」していた
- 読んでも忘れる、整理されない

**YouTube始めた後**

- 「これ動画にできるかも」フィルターが常にON
- メモの取り方、整理の仕方が変わった

### 3. 具体的な情報源（1:00-2:00）

アウトプット前提で見ると、こんな場所がネタの宝庫になる：

| 情報源 | 特徴 |
| --- | --- |
| arXiv | 最新研究、プレプリント |
| GitHub | 実装・コード、トレンド |
| MIT Tech Review / 日経サイエンス | 一般向けに噛み砕かれた科学記事 |
| 教科書・サーベイ | 基礎の再発見 |

**ポイント**: 難しい論文を読む必要はない
- これらはプロ向けのプラットフォームだけど、それにこだわる必要はない
- 中高の宿題の変わった解き方でも価値がある
- 誰かの勉強の役に立つかもしれない

### 4. 差別化チェック（2:00-2:30）

- YouTube先行動画を調査
- 独自色を出せるか判断
- 被ったらお蔵入りも選択肢（実際にいくつかある）

### 5. まとめ（2:30-3:00）

- アウトプットを前提にすると、世界の見え方が変わる
- 完璧なインプットを待たなくていい
- まず「伝えたい」から始めてみよう
