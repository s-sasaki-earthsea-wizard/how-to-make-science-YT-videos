# 5-1: 多言語展開の意義（TTS用）

## TTS用脚本

### 1. 導入（0:00-0:30）

[friendly tone] このチャンネルでは、英語、中国語、韓国語など、複数言語の字幕と吹き替えを用意しています。

[curious] 「なんでわざわざそんな手間をかけてるの？」と思う方もいるでしょう。

今日は、その理由についてお話しします。

---

### 2. 理由その1：実利（0:30-1:30）

[matter-of-factly] ひとつ目の理由は、シンプルに視聴者を増やしたいから。

日本語を話す人は、世界で約1億2000万人です。
一方、英語は15億3000万人、中国語は14億7000万人。

単純計算で、見てもらえる機会が何倍にもなるわけです。

実際、このチャンネルの視聴者の15パーセントは、日本以外の国からのアクセスです。
特に台湾や韓国からの視聴者が多いですね。

[calm] 多言語対応は手間がかかりますが、それだけのリターンはあると感じています。

---

### 3. 理由その2：理想（1:30-2:30）

[slightly shy] ふたつ目の理由は、ちょっと大げさかもしれませんが、少しでも世界平和に貢献したいから。

国同士は、いろいろ複雑な問題を抱えていますよね。

[heartfelt] でも、科学は普遍的なものです。
言葉や国境を超えて、人と人をつなげる力があると信じています。

[warmly] 科学動画を通じて、異なる国の人たちが同じ知識を共有できる。
そんな小さな架け橋になれたら嬉しいな、なんて思っています。

[soft chuckle] こういうことを真面目に言うのはちょっと恥ずかしいですけどね！[laugh]

---

### 4. まとめ（2:30-3:00）

[calm] というわけで、多言語展開をしている理由は2つ。
視聴者を増やしたいという実利と、世界平和への貢献という理想。

[warmly] 手間がかかっているのは事実ですが、この2つのバランスをとりながら、これからもチャンネルを運営していきたいと思います。

---

## 使用したAudio Tags

| タグ | 使用箇所 | 意図 |
|------|----------|------|
| `[friendly tone]` | 冒頭 | 親しみやすい導入 |
| `[curious]` | 疑問の提示 | 視聴者の疑問を代弁 |
| `[matter-of-factly]` | 実利の説明 | 事実を淡々と伝える |
| `[calm]` | リターンの話、まとめ | 落ち着いた語り |
| `[slightly shy]` | 世界平和の話題 | 照れながら切り出す |
| `[heartfelt]` | 科学の普遍性 | 心からの信念 |
| `[warmly]` | 架け橋の話、締め | 温かみのある語り |
| `[soft chuckle]` | 恥ずかしい発言 | 照れ笑い |
